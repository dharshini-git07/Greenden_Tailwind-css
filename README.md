🌿 Greenden - Responsive Plant Store Website

Greenden is a responsive multi-page website built using **HTML5** and **Tailwind CSS**. The project showcases a clean, modern, and mobile-friendly user interface for a plant store while demonstrating the use of Tailwind's utility-first CSS framework.

 Features

- Responsive Navigation Bar
- Home Page with Hero Section
- About Us Section
- Best Seller Products
- Customer Reviews
- Newsletter Subscription
- Products Page with Search Bar
- Responsive Product Grid
- Contact Page with Responsive Contact Form
- Sticky Navigation Bar
- Mobile-Friendly Design

Technologies Used

- HTML5
- Tailwind CSS

📱 Responsive Design

The website is fully responsive and optimized for:

- 📱 Mobile Devices
- 💻 Laptops
- 🖥️ Desktop Screens

Tailwind responsive utilities such as `sm:`, `md:`, and `lg:` are used to create adaptive layouts.
